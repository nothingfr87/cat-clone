# Cat Clone

A Clone of the linux `cat` command made in c

---

## Descripition

A very very very simple `cat` command clone in c, uses the File I/O methods to read data from the given file, then displays it

---

## How to run

simply clone the repositery then run `make all`

```bash
# Cloning the repositery
git clone https://github.com/nothingfr87/cat-clone.git
cd cat-clone/

# Build the source file
make build

# Run the file
./cfetch
```
